# Playwright QA Portfolio — PetStore

## Overview
This repository contains automated end-to-end tests created with Playwright for the PetStore application.

## Scope
The project covers:
- login
- registration
- search
- cart

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Project Structure
- `tests/` — test files
- `pages/` — Page Object Model files
- `test-data/` — test data
- `docs/` — additional documentation
