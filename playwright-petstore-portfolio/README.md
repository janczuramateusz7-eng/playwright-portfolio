Overview
This repository contains automated tests created using Playwright.
The project simulates real QA work and includes automated end-to-end testing scenarios.

Scope
Tested areas:

Sign In
Sign Up
Search
Cart

Tech Stack

Playwright
JavaScript

Test Execution

Run all tests:

npm test

Run tests with browser:

npm run test:headed

Run UI mode:

npm run test:ui
